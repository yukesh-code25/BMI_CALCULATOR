# 🧮 BMI Calculator

A simple and responsive **BMI (Body Mass Index) Calculator** built using **HTML, CSS, and JavaScript**. This project allows users to enter their height and weight, calculate their BMI instantly, and view their weight category based on the result.

---

## 📌 Project Overview

The BMI Calculator is a beginner-friendly web application that helps users determine whether they are underweight, have a normal weight, are overweight, or are obese based on their height and weight.

This project demonstrates the use of **HTML for structure**, **CSS for styling**, and **JavaScript for calculations and DOM manipulation**.

---

## ✨ Features

* 📏 Input height and weight
* ⚡ Instant BMI calculation
* 📊 Displays BMI value
* 🩺 Shows BMI category:

  * Underweight
  * Normal Weight
  * Overweight
  * Obese
* 📱 Responsive design
* 🎨 Clean and user-friendly interface

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
